## USAGE

### ECHO-SERVER and CLIENT
In `src/echo-server-and-client`
```
$ make
$ ./server 7000

# another terminal
$ ./client localhost 7000
```

### SIMPLE-ECHO-SERVER with telnet
In `src/simple-echo`
```bash
$ make
$ ./simple_echo_server

# another terminal
$ telnet localhost 7000
```

### SIMPLE-HTTP-SERVER
In `src/http`
```bash
$ make
$ ./http-server
```
Access http://localhost:8080/
